axffmpeg.spi
============

Susie plugin to invoke external FFmpeg, in order to handle a movie file as an image container.

*THIS REPOSITORY DO NOT AND WILL NOT INCLUDE FFmpeg ITSELF*. You need to get it somewhere else.
